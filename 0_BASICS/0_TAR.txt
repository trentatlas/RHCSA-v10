

- Create TAR

-f  File name type of the archive file

tar -cvf student.tar /home/student
tar -cvzf student.tar.gz /home/student
tar -cvzf student.tgz /home/student
tar -cvjf student.tar.bz2 /home/student


-C restore into another DIR
tar -xvzf student.tgz  -C /home/student/RESTORE/
